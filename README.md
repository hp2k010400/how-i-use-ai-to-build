# How I Use AI to Build Faster

This repository shows how I use AI tools (ChatGPT, Claude Code) as a core part of my development workflow.

Rather than using AI just for quick answers, I use it to:
- plan features
- debug issues
- refine and improve code
- understand unfamiliar systems
- ship working solutions faster

---

## 🧠 My Approach

I treat AI as a development partner, not a replacement.

My workflow typically looks like:

1. Define the problem clearly  
2. Use AI to explore solutions or identify issues  
3. Test and validate outputs  
4. Adapt the result to fit the real project  
5. Iterate quickly
